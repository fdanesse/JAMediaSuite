En este capítulo, se intenta explicar lo básico para que puedas comenzar a programar en python. La idea es que puedas escribir un pequeño programa en un archivo y luego puedas ejecutarlo.


¿Qué es programar?:

Es hacer una lista de instrucciones que mi computadora debe ejecutar.
Para ello lo que hacemos es escribir esas instrucciones en un archivo y luego decirle a la computadora que lea ese archivo y ejecute las instrucciones que allí se encuentran escritas.

Pero los programas se escriben en un lenguaje de programación y por lo tanto hay que aprender uno de esos lenguajes. En esta ayuda aprenderás a programar en python.

Los programadores trabajan en editores de código, que en realidad son editores de texto con funciones especiales para la tarea que deben realizar.
JAMediaEditor es un editor de código orientado a programar en python, pero puedes utilizarlo para cualquier otro lenguaje de programación que conozcas.


Los Interpretes:

Los interpretes son interfases capaces de entender (interpretar), determinados lenguajes.
Por ejemplo, bash es el interprete de comandos de GNU/linux. Cuando abres la terminal, abres el interprete de bash. Esto quiere decir que la terminal responderá a cualquier comando de bash pues sabe interpretarlo.

python es un lenguaje interpretado, por lo tanto, tiene su propio interprete.
Si en la terminal escribes python y das enter, entrarás en el interprete de python. Te darás cuenta de esto porque el promp mostrará los simbolos ">>>" indicandote que espera instrucciones python.
Para salir del interprete de python debes hacer control + d.

Más adelante veremos como utilizar el interprete de python directamente, ahora nos centraremos en escribir un programa en un archivo y ejecutarlo.

Los programas python se escriben en archivos de texto con la extensión “py” y luego, para ejecutar ese archivo, debes decirle al interprete de python que lo haga (python miarchivo.py).

Es común que cuando estás aprendiendo a programar en python, intentes ejecutar un archivo y escribas python olvidando poner el nombre del archivo que deseas ejecutar, con lo cual entrarás en el interprete de python, recuerda entonces, que para salir del interprete, debes hacer control + d.


Escribir un programa en python:

Utilizando JAMediaEditor, en un archivo nuevo, escribe lo siguiente:

print("Hola")

Luego guarda ese archivo con el nombre "programa" y la extensión ".py" (programa.py)
Si ejecutas ese archivo, en la terminal imprimirá "Hola".

Este podríamos considerarlo como el programa python más básico que podemos construir.
En él se ejecuta la función de python llamada print que imprime en pantalla el texto contenido entre paréntesis. Las comillas que rodean al texto le indican a python que esos datos son de tipo string (cadena de caracteres) y deben tomarse textualmente y no como una orden o función python.
Sin esas comillas, python trataría de interpretar ese texto como una función y dado que Hola no es una función de python, tu programa fallaría.

Los errores en los programas se denominan bugs.

Los programas son en definitiva una serie de ordenes escritas en un archivo.
Cada orden se escribe en un renglón.
El interprete de python interpreta cada renglón en el orden en que están escritos desde arriba y hacia abajo.
Cualquier error (bug) que contenga el programa, hará que la ejecución del mismo se detenga en la linea donde se encuentra el error.
Si ejecutas un programa que contiene un error, python detendrá su ejecución y te indicará en que línea se encuentra el error y que tipo de error es.

Por ejemplo, si mi programa dijera:

print(Hola)

fallaría y en la terminal me aparecería el siguiente mensaje:

File "/home/flavio/Documentos/miprograma.py", line 1, in <module>
print(Hola)
NameError: name 'Hola' is not defined

Como no puse comillas alrededor de la palabra Hola, el interprete de python da error ya que intenta ejecutar esa palabra como si fuera una función
(python no sabe como interpretar este dato).

Aprender a programar en python es en definitiva conocer las funciones de python y los tipos de datos que cada una de ellas es capaz de interpretar y manejar.

Todas las funciones de python constituyen lo que llamamos el API del lenguaje y se encuentran documentadas en la página web del lenguaje: https://www.python.org/, algunas de esas funciones las iremos explicando poco a poco en esta ayuda a medida que las vayamos necesitando para desarrollar los diferentes ejemplos planteados.

Los tipos de datos que python es capaz de manejar, será el tema central del próximo capítulo de esta ayuda.
